DELETE /api/customer/:idInternalCustomer

Description

Delete a Customer from the system.

Parameters

idInternalCustomer required, type: string

Declare in URL. The Internal Customer Number of the requested Customer.

Resource Information

API Version 1.0
HTTP Method DELETE
Login Required true
Response Format application/json
