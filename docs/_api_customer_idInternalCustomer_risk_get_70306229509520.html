GET /api/customer/:idInternalCustomer/risk

Description

Get a Risk Analysis, also known as Calculated Risk, for a given Customer.

Headers

Content-Type

application/json

x-access-token

Parameters

idInternalCustomer required, type: string

Declare in URL. The Internal Customer Number of the requested Customer.
