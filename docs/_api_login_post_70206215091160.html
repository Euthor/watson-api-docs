POST /api/login

Description

Login and receive an access token to be used as an x-access-token header in subsequent requests. Ensure the requested user has the appropriate priviledges for subsequent requests.

Headers

Content-Type

application/json

Parameters

username required, type: string

The username of the user.

password required, type: string

The password of the user.

Example Request

{
    "username": "Administrator",
    "password": "sample-password-Sx123"
}

{
    "success": true,
    "token": "eyJhbGciOiJIUzI1NiJ9.YWRtaW5pc3RyYXRvciQyYSQwNCQycXlNL05hZDYudkN1cXpsRjNibkguRnZTZS9MRHNRRmFuNUpwNzd5cGo3QmsuM3pqUURmaQ.M1qhGoR-LBtJI4WnSjxLLy-OzofSCqmFhcM9QdTdM7Y"
}
