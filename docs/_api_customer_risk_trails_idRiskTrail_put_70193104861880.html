PUT /api/customer/risk-trails/:idRiskTrail

Description

Update a Risk Trail. Used to approve a Risk Trail.

Headers

Content-Type

application/json

x-access-token

Parameters

idRiskTrail required, type: integer

Declare in URL path. The unique numeric identifier of the Risk Trail.

idApprovalStatus required, type: integer

Declare in request body. 1 = Approved, 2 = Rejected.

idRiskType required, type: integer

Declare in request body. 1 = Low, 2 = Medium, 3 = High, 4 = Extremely High.

comments required, type: string
