POST /api/customer

Description

Create a new Customer. Responds with the unique identifiers of the created Risk and Audit Trails.

Headers

Content-Type

application/json

x-access-token

eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9

Parameters

main required, type: object

Declare in request body. The main details of a Customer. See sample request.

details required, type: object

Declare in request body. The details of the Customer. See sample request.

{
    "details": {
        "dateBirth": "1965-02-01",
        "firstName": "John",

        "idCountryNationality": 137,

    "main": {
