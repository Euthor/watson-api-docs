Error Codes

HTTP Status Codes

The API attempts to return appropriate HTTP status codes for every request.
Code Text Description
200 OK Success. There is response data as well.
204 No Content Success. There is no response data.
400 Bad Request The request was invalid (possibly because of validation errors) or cannot be otherwise served.
401 Unauthorized Request The x-access-token is not valid.
500 Server Error The server encountered an error.
